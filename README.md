# AI-Based Volume Calculator

## Description
This project calculates the volume of a **Slanted Elliptical Cylinder**, an irregular solid.

## Language Used
C#

## Formula
V = π × a × b × h

## How it Works
The program asks the user to input:
- semi-major axis (a)
- semi-minor axis (b)
- height (h)

It then computes and displays the volume.
